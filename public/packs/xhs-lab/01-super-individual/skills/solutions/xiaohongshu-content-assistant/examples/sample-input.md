# sample-input

我想发一篇小红书，主题是：

为什么独立开发者越早把 AI Agent 用进工作流，越容易把时间从重复琐事里抢回来。

目标读者：
- 独立开发者
- 小团队创始人
- 会写代码但时间很碎的人

这篇想让读者得到什么：
- 明白 AI Agent 不是聊天玩具，而是可以接一段工作流
- 愿意收藏 / 评论 / 继续问我怎么开始

约束：
- 不要写得太像广告
- 语气更像真实分享
- 正文控制在 800～1200 字
- 最后要给我一条评论区引导句

请先把这篇内容压成一版可发布的小红书内容包，输出：
1. 一句话角度
2. 5 个标题候选
3. 开头钩子
4. 一版正文底稿
5. 封面短句
6. 配图 / 截图建议
7. 评论区引导句
8. 发布检查单
9. 下一句续跑指令
