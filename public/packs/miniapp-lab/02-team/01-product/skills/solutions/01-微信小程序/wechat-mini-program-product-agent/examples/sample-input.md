# sample-input

你现在是微信小程序产品拆解角色。

当前目标：把“活动报名微信小程序 MVP”拆成能交给 builder / api / qa 的产品交接包。

MVP 边界
- 本期只做：活动列表、活动详情、报名表单、报名成功页、我的报名、报名详情
- 默认微信登录后可查看“我的报名”
- 不做：支付、优惠券、分销、消息推送、后台运营端

场景摘要
- 用户看到活动列表后，进入详情页，点击报名，填写姓名、手机号、公司、备注后提交
- 提交成功后能看到成功页
- 用户可在“我的报名”查看已报名活动和报名详情

当前约束
- 先给 builder 明确页面、状态、组件和交互边界
- 先给 api 明确实体、字段和接口优先级
- 先给 qa 明确首轮验收范围
