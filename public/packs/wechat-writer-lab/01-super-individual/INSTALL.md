# 01-super-individual 安装说明

> 这个包适合：你先把一个主题压成一篇像样的公众号文章草稿，不先上来做复杂内容体系。

---

## 👀 这个包会帮你产出什么

跑完后，你最少应该拿到：
- 文章切口
- 标题候选
- 文章大纲
- 导语与正文初稿
- 小标题 / 配图建议
- 结尾 CTA
- 发布检查单

---

## ⚡ 最短用法

### 1）安装
```bash
bash ./install_to_profile.sh <profile-name-or-path>
```

### 2）直接试跑
```bash
hermes -p <your-profile> chat --skills wechat-official-account-writer -q "$(cat skills/solutions/wechat-official-account-writer/examples/sample-input.md)"
```

---

## ✅ 跑完后你重点看什么

不要只看它有没有写稿，重点看这 4 件事：
- 有没有先定清楚文章切口
- 有没有给 3～5 个标题备选
- 有没有把大纲、导语和正文初稿写出来
- 有没有把结尾 CTA 和发前动作补上
